🛡️ VeriFact-IN: AI-Powered Fact Checker
VeriFact-IN is an advanced NLP tool designed to verify the authenticity of news and detect AI-generated text. This tool supports both Hindi and English languages.

🚀 Key Features
Real-time News Analysis: Instantly determine whether a news piece is real or fake.
AI vs Human Detection: Identify whether a text was written by a human or generated by AI.
Bilingual Support: Support for both English and Hindi text input.
Minimalist UI: A clean and fast interface built on Streamlit.

🧠 Technical Architecture
The core of this project is a Zero-Shot Classification model.
Model: facebook/bart-large-mnli (BART model by Meta).
Logic: This model understands the context of text without any additional training.
NLI: It uses Natural Language Inference to calculate a confidence score.

🛠️ Tech Stack
Language: Python 3.13
Libraries: Transformers (Hugging Face), Streamlit, PyTorch.
